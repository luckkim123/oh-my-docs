**Goal:** repo-docs(README·CHANGELOG·community health files) 장르를 카드+루브릭으로 편입하고, 그 첫 소비자가 요구하는 공통 인프라 일반화(§4.4 13항)를 수행한 뒤, omd 자신의 README/CHANGELOG를 신규 파이프라인으로 재생성(dogfooding)해 v0.2.0을 출하한다.

**Architecture:** spec `docs/superpowers/specs/2026-07-11-omd-program-design.md` §5 R2 행 + R2 build-order constraints가 SSOT. 관통 패턴은 "스킬/에이전트 본문의 오피스 하드코딩 → **카드가 정의한다**로 위임"(D1 — 본문에 포맷별 if/else 금지). 순서: output-layout(D4) → verify 훅(D5) → 뒷단 일반화 묶음(§4.4-3~7, 인접 커밋) → repo-docs 카드+루브릭 → 앞단 일반화(§4.4-8) → route/tests(§4.4-9) → dogfooding.

**Tech Stack:** Python 3 stdlib(훅·테스트), pytest, 순수 Markdown + 빌린 린터 체인(markdownlint-cli2 via npx — 선택적, D3 degrade), git.

### 3.3 Artifact-set snapshots and the manifest

- Snapshot = copy the **whole** `outputs/<slug>/current/` directory to

# ── D5 (R2): md 계열 Edit|Write 트리거 ────────────────────────────────
# md 산출물(repo-docs·site)은 Bash가 아니라 Edit|Write로 태어난다. 발화 조건 3중:
# .md 확장자 AND outputs/<slug>/ 경로 AND .omd/<slug>/ 실재(slug 컨텍스트 — §7 ②).
# 작업 메모(.omd/<slug>/*.md)는 deliverable이 아니므로 침묵(§7 ⑥ alert fatigue).

def run_md_hook(file_path: str, tool_name: str = "Edit", cwd: Optional[str] = None) -> str:
    payload = {"tool_name": tool_name, "tool_input": {"file_path": file_path}}
    if cwd is not None:
        payload["cwd"] = cwd
    proc = subprocess.run(
        [sys.executable, str(HOOK)],
        input=json.dumps(payload),
        capture_output=True, text=True,
    )
    assert proc.returncode == 0, f"hook exited {proc.returncode}: {proc.stderr}"
    return proc.stdout


def _with_slug(tmp_path, slug="2026-07-13-omd-readme"):
    (tmp_path / ".omd" / slug).mkdir(parents=True)
    return slug


def test_md_edit_fires_and_arms_on_single_file(tmp_path):

def _concrete_refs(text):

def _readme_h2s():
    return [m.group(1).strip() for m in re.finditer(r"^## (.+)$", README, re.MULTILINE)]


def test_readme_required_sections_in_preset_order():
    """gate ①: library 프리셋 필수 섹션 존재 + 순서 (Install → Usage → … → License 마지막)."""
    heads = _readme_h2s()
    for required in ("Install", "Usage", "License"):
        assert any(required in h for h in heads), f"README: required section {required!r} missing"
    idx = {name: next(i for i, h in enumerate(heads) if name in h)
           for name in ("Install", "Usage", "License")}
    assert idx["Install"] < idx["Usage"] < idx["License"]
    assert "License" in heads[-1], "License must be the last section"


def test_readme_short_description_and_toc_rule():
    """gate ①: 제목 직후 짧은 설명(≤120자 한 줄) + 100줄 이상이면 ToC 필수."""

            "README ≥100 lines requires a ToC (standard-readme)"

def test_changelog_unreleased_first_and_semver_descending():
    """gate ⑤: [Unreleased] 최상단 + semver 섹션 ISO 날짜 + 역순."""

    assert entries, "no dated semver sections found"
    versions = [tuple(map(int, e)) for e in entries]
    assert versions == sorted(versions, reverse=True), f"not descending: {versions}"

    assert types <= allowed, f"non-keepachangelog types in semver region: {types - allowed}"
